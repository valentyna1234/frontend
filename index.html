This is a list of commands for working with the terminal that we covered in the lesson:

  • pwd (used this command during homework)
  • git status (used this command during homework)
  • git clone (used this command during homework)
  • (used this command during homework)
  • git add (used this command during homework)
  • git commit -m"a-comment-to-the-commit"(used this command during homework)
  • git push (used this command during homework)
  • git branch (shows which branch we are in)
  • git checkout -b "branch-name" (create and switch to new branch)
  • git checkout "the-name-of-an-existing-branch" (switch to existing branch)
  • git merge "branch-name" (merge branches)
  • git rebase (takes all commit history we are behind from the "master" branch in the branch we are in and tries to merge all the commits from both branches
  • git pull (pulls commits that occurred in the remote repository to the local repository)
  • git fetch (updates information on the remote repository)
  • git log (lists the commits made in the repository)
  • git reset --"argument(soft/hard) location of the commit (for example HEAD)~number of steps (for example 1)" (cancel commit but save changes, "for example" - in this case, the changes will be applied to the last commit)
  • git stash ("put changes in a pocket" (all local changes will be hidden)
  • git stash apply (re-add the changes that were hidden)
  • git revert "commit id" (allows to cansel a specific commit, used this command during homework)
  • git cherry-pick (selectively applies commits to the current working HEAD branch, can pick a commit from one branch and apply it to another.
